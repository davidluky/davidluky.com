MMX Phase 4 P3 — Weapon cycling (authentic password unlock)

Cold-boot → intro-skip → title → PASSWORD → enter 8441 2136 4421 (all 8 Mavericks, 4 armour parts, 8 heart tanks, 3 sub-tanks) → stage-select replay → enter stage → cycle weapons via pause-menu → capture CGRAM + screenshot. This is the "Path 1" replacement for the withdrawn synthetic-bit-flip attempt.

Authentic unlocks verified. All 14 datacrystal unlock-bytes match. X has full armour (helmet/chest/boots visible in every shot). Weapon icon in the HUD changes between cycles.
Honest caveats.
  • Stage is Boomer Kuwanger (cursor 7), not Chill Penguin (cursor 8). Post-password stage-select cursor starts at 9 (Dr. Light center), not at 1 like the fresh-boot stage_select.mss fixture used in Session 22 — and the 40-tap REPLAY from that session never hits cursor 8 from this starting state. Fixing that is follow-up work. What you're looking at is the authentic all-unlocks state in a Maverick stage, just not the one I aimed for.
  • SELECT does nothing for weapon cycling in MMX1. That's the bug the first P3 attempt hit silently. Real cycling requires START → pause menu → D-pad → START to resume. This page uses that flow.
  • Cycle success is partial. Out of 9 attempted cycles, 3 (c02, c07, c09) produced a real CGRAM diff in row 9 and visible X body-colour change. The rest landed on the same weapon (menu doesn't wrap past the end of the acquired- weapon list, so repeated DOWN beyond last weapon is a no-op). Still more information than the previous synthetic-bit-flip attempt which never cycled at all.
